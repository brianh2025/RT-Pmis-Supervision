① 使用者認証流程
Login.jsx → AuthContext.signIn()
→ supabase.auth.signInWithPassword()
→ Supabase Auth 驗証帳密
→ 回傳 JWT Token + Session
→ ProtectedRoute 放行 → Dashboard
② 一般 CRUD 資料流
頁面元件 觸發操作
→ useProjects() / 直接 supabase.from()
→ Supabase JS SDK 組裝 REST 請求
→ PostgreSQL + RLS 政策驗証
→ 回傳 data / error → React 重新渲染
③ Google Drive 日誌同步
DriveSyncModal 觸發同步
→ fetch(EDGE_FN_URL, { mode:'sync_one' })
→ sync-diary Edge Function 執行
→ Drive API 下載 Excel → fflate 解析
→ upsert daily_logs / report_items
